Time incorrect?

If the Sehri and/or Iftar time appear to be incorrect.
You can change the calculations used underSettings.
You can choose from one of the predefined values or set these to Other and use the provided sliders to fine tune the calculations.

How do I set a custom alarm?

To add a custom ringtone, put your MP3 in the Alarms folder under internal storage (some devices may call this /storage/sdcard0). If the folder does not exist you can create it.

How do I set my location?

Even though detecting your location should not be a problem (it should be detected automatically), there is an option to manually set your location just in case you have any issues.
Simply click SET LOCATION in the action bar and type your location.
You will be presented with a list of suggestions; simply select one.
